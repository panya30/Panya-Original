### Without Skills (Memory Only)
- Panya remembers everything
- But can't DO anything special
- Just a memory bank

### Without Brain (Skills Only)
- Can do many tasks
- But forgets everything between sessions
- No relationship, no identity

### With Both
- Panya remembers you (Brain)
- Panya knows how YOU work (Skills learned from you)
- Panya can do tasks YOUR way (Personal Skills)
- Panya grows smarter over time (Brain + Skills evolve)
- Panya can share knowledge (Export Skills + Knowledge)

## Technical Stack

### Current (Phase 1)
```
- Runtime: Bun
- AI: Claude (Anthropic)
- Local DB: SQLite + FTS5
- Vector: ChromaDB
- Protocol: MCP
- Skills: Claude Skills format (SKILL.md)
```

### Future (Phase 2+)
```
- Auth: Clerk or Supabase Auth
- Cloud DB: Supabase (Postgres + pgvector)
- Graph: SQLite (lightweight) or Neo4j (advanced)
- Sync: Supabase Realtime
- Mobile: React Native or PWA
- Desktop: Tauri
```
