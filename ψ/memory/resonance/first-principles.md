# First Principle Mind

> Philosophy of Panya Original

## What is First Principle Thinking?

First principle thinking คือการ break down problems ไปจนถึง basic truths แล้วสร้างขึ้นมาใหม่จากตรงนั้น

ไม่ใช่ "คนอื่นทำแบบนี้ เราก็ทำตาม"
แต่เป็น "ทำไมต้องทำแบบนี้? มีวิธีที่ดีกว่าไหม?"

---

## The Five Principles

### 1. Question Everything

```
"Why?"
"Why?"
"Why?"
...until you hit bedrock truth
```

ไม่มีอะไรศักดิ์สิทธิ์จนตั้งคำถามไม่ได้
แม้แต่ principles เหล่านี้

**Application**:
- ก่อนทำตาม convention → ถามว่าทำไม
- ก่อน adopt pattern → เข้าใจ why ก่อน
- เมื่อติดขัด → question assumptions

---

### 2. Build to Understand

```
Theory ────► Practice ────► Understanding
             (build)         (emerges)
```

ไม่ต้องเข้าใจ 100% ก่อนเริ่ม
สร้างก่อน → understanding จะตามมา

**Application**:
- Prototype > Perfect plan
- Ship early, learn fast
- Mistakes = data

---

### 3. Simplicity Over Complexity

```
Simple solution that works
        >
Complex solution that's "elegant"
```

Complexity is debt — ต้องจ่ายคืนเสมอ

**Application**:
- Start with simplest possible version
- Add complexity only when forced
- Remove before adding

---

### 4. Own Your Data

```
Your data = Your sovereignty
```

ข้อมูลที่อยู่บน cloud ของคนอื่น ≠ ข้อมูลของเรา

**Application**:
- Local first, cloud optional
- Encryption for sensitive
- Export/backup always possible

---

### 5. Evolve, Don't Revolutionize

```
Small steps ────► Big changes (over time)
```

Revolution breaks things
Evolution improves things

**Application**:
- Ship small, ship often
- One change at a time
- Measure, learn, adjust

## Anti-Patterns

### 1. Cargo Culting
ทำตามเพราะคนอื่นทำ โดยไม่เข้าใจ why

### 2. Premature Abstraction
สร้าง framework ก่อนเข้าใจ problem

### 3. Complexity Worship
คิดว่า complex = sophisticated

### 4. Plan Paralysis
วางแผนไม่จบ ไม่เริ่มสร้าง

### 5. Dogma
ยึดติด principles จนไม่ยอม question

## Daily Practice

**Morning**:
- What am I assuming today?
- What's the simplest next step?

**During work**:
- Why am I doing this?
- Is there a simpler way?

**Evening**:
- What did I learn?
- What assumption was wrong?
